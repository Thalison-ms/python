- Tudo em Python é objeto
- Pacote em Python é um conjunto de scripts, onde cada script é uma função
- Python é case-sensitive, ou seja, maiusculo e minusculo faz diferença
- Listas podem ser concatenadas em Python
- Em Python existe a sobrecarga de operadores
- São imutáveis (str, bool, float, int, long, complex, tuple)
- A indexação é exclusiva, ou seja, retorna até a posição anterior informada, exemplo a = "Babilonia", print(a[:3]) irá retornar "Bab"
- Variáveis em Python tem tipagem Dinâmica, ou seja, a = 5 pode se tornar a = "Teste"
- Nomenclaturas de variáveis não podem começar com número
- Não é permitido o uso de palavras reservadas para nomear variáveis
